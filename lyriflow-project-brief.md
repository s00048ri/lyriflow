# LyriFlow — 歌詞テレプロンプター プロジェクトブリーフ

## 概要

バンドのボーカリスト向け歌詞テレプロンプターWebアプリ。iPadのブラウザ上で動作し、Bluetooth HIDペダル（PageDown/PageUp等を送信）でスライド送りしながら練習・ステージ演奏で使用する。

## 背景・動機

- 歌詞が覚えられないので、iPadで歌詞をスライド表示してペダルで送っている
- 既存アプリ（Setlists by Bombing Brain等）を試したが：
  - **曲と曲の間でリスト画面に戻る必要がある**（両ペダル同時押しで次曲に行く仕様だが、手持ちのデバイスでは動作しなかった）
  - **頻繁にクラッシュする**（ステージ上での信頼性に不安）
- 既存アプリは多機能すぎる（コード譜、移調、MIDI、ChordPro形式など不要な機能が多い）
- 「歌詞テキストを貼り付けたら、すぐスライド表示でき、ペダルだけでセットリストを最後まで通せる」シンプルなアプリが欲しい

## 技術スタック

- **Webアプリ（HTML/CSS/JS、単一ファイルまたはシンプルな構成）**
- iPad Safariで動作（`apple-mobile-web-app-capable` 対応）
- 将来的にPWA化も視野に入れるが、まずはプロトタイプ優先
- ネイティブiOSアプリ化は将来の選択肢として保留

## コア要件

### 1. 歌詞入力（編集モード）
- テキストエリアに歌詞を貼り付ける
- **空行で区切ってセクション（スライド）に自動分割**
  - 例：Aメロ4行 → 空行 → Bメロ4行 → 空行 → サビ → 各ブロックが1スライド
- 【Aメロ】【サビ】のようなカッコ付きテキストはセクションラベルとして表示（任意）
- 曲名を入力できる

### 2. 演奏モード（スライド表示）
- **1画面に1ブロック（スライド式）** ※スクロール式ではない
- ペダル操作：
  - **PageDown / → / Space / ArrowDown** → 次のスライド
  - **PageUp / ← / ArrowUp** → 前のスライド
  - 最後のスライドで「次」を押すと → **自動的に次の曲の最初のスライドへ**（これが最重要要件）
  - 最初のスライドで「前」を押すと → 前の曲の最後のスライドへ
- 曲名と進捗（例：3/12）を常時表示
- プログレスバー
- ステージ向けのダーク背景、大きなフォント
- 画面の自動ロック防止（`navigator.wakeLock` API）

### 3. セットリスト管理
- 複数曲を登録してセットリストとして並べる
- 曲順の並べ替え
- 曲の追加・削除
- セットリスト内の曲をタブ等で表示・切り替え

### 4. データ保存
- ファイル書き出し（JSON）/ 読み込みで永続化
- localStorage も併用して作業中のデータを保持

### 5. フォントサイズ
- 演奏モードでフォントサイズを調整可能
- 設定を記憶する

## UIデザイン方針

- **ダークテーマ**（ステージ照明に干渉しない）
- 背景：ほぼ黒（#08080d 〜 #0a0a0f 系）
- テキスト：明るい白系（#e8e6f0）
- アクセント：オレンジ系（#ff6b3d）
- フォント：日本語歌詞が多いので `Noto Sans JP` + 英語UI部分に `Outfit` 等
- 演奏モードは極限までシンプルに（余計なUIを排除、歌詞とペダル操作に集中）
- 編集モードも直感的に（テキストを貼り付けて曲名を入れたら、すぐ演奏モードへ）

## 演奏モードのスライド遷移動作（最重要）

```
曲1: [スライド1] → [スライド2] → [スライド3]
                                        ↓ PageDown
曲2: [スライド1] → [スライド2] → [スライド3] → [スライド4]
                                                    ↓ PageDown
曲3: [スライド1] → ...
```

- ペダルを踏み続けるだけで、セットリスト全体をシームレスに通せる
- 曲が切り替わるタイミングで、短いトランジション（曲名表示など）があると良い

## 既存アプリ調査結果（参考）

| アプリ | 良い点 | 不満点 |
|---|---|---|
| **Setlists** (Bombing Brain) | スライド式、ペダル対応、無料 | 曲間でリスト画面に戻る、両ペダル同時押しが動かない場合あり、クラッシュ頻発 |
| **OnSong 2026** | 超多機能、ペダル対応 | サブスク($4.99/月)、機能過多で複雑 |
| **LivePrompter 2** | 無料、クロスプラットフォーム | スクロール方式中心、ChordPro必須、外部エディタ必要 |
| **BandHelper** | 自動曲遷移あり | 設定が非常に複雑 |
